Filosofia
de Design

Engenharia Industrial Criativa — onde o design auxilia na clareza do pensamento e na detecção de falhas.

// 01

Neo-Brutalismo Industrial

Bordas sólidas de 4px, sombras hard offset de 8×8px, tipografia pesada uppercase. Um estilo que prioriza legibilidade e hierarquia visual, ideal para engenheiros que preferem interfaces diretas sem distrações decorativas.

// 02

Arquitetura Modular

Plugin architecture onde o core gerencia apenas navegação e estado global. Cada um dos 11 módulos é independente — pode ser executado isoladamente ou integrado à suíte. Lazy-loading para performance máxima.

// 03

Software Livre

Distribuído sob GPLv3. Diferente de licenças permissivas, a GPLv3 garante que o ProEng permaneça livre para sempre — liberdade de usar, estudar, modificar e distribuir, mantendo essas liberdades para outros.

// 04

Persistência Total

Estado completo de todos os módulos serializado em JSON no formato .proeng. Salvar, carregar, versionar. Cada diagrama, cada corrente, cada card — tudo preservado com fidelidade absoluta.

// 05

14 Temas Dinâmicos

Troca de tema em tempo real em todos os módulos, diagramas, toolbars e diálogos simultaneamente. Do Neo-Brutal ao Dracula — paridade funcional completa em todos os 14 esquemas visuais.

// 06

Export de Alta Resolução

Exportação em PNG com 3× supersampling para qualidade de publicação e PDF A3 landscape para relatórios técnicos profissionais. Cada diagrama, pronto para apresentação.

⚡
Lazy Loading

Módulos instanciados somente quando acessados. Startup rápido mesmo com 11 módulos pesados.

🎨
T() Singleton

Acesso ao tema ativo via T()["accent"]. refresh_theme_recursive() propagado para todos os widgets.

💾
Persistência JSON

Formato .proeng: snapshot completo de todos os módulos. get_state() / set_state() em cada módulo.

📐
Vector Rendering

QGraphicsScene com aceleração de hardware. Gradientes dinâmicos, sombras e geração de polígonos em tempo real.

🔌
BaseModule

Interface abstrata: get_state, set_state, refresh_theme, get_view. Contrato que garante integração perfeita.

📤
Export Engine

PNG 3× supersampling para publicação. PDF A3 landscape para relatórios técnicos profissionais.

Referência
de API

Classes e interfaces dos módulos principais. Toda a API usa Python 3 com PyQt5.

Sistema de Temas
from proeng.core.themes import T, set_theme # Acessar cor do tema ativo color = T()["accent"] # #2563EB bg = T()["bg_app"] # #050816 line = T()["line"] # #38BDF8 # Trocar tema set_theme("synthwave") set_theme("dracula")
T() Retorna dicionário de cores do tema ativo
set_theme(name) Altera tema ativo. Aceita qualquer dos 14 slugs
cycle_theme() Avança para o próximo tema na sequência
refresh_theme_recursive(w) Propaga atualização para todos os widgets filhos
Flowsheet — PFD
from proeng.modules.flowsheet import FlowsheetWidget widget = FlowsheetWidget() # Executar balanço de massa (Kahn) widget.solve_mass_balance() # Persistência state = widget.get_state() # {nodes, edges, streams} widget.set_state(state) # Exemplo industrial widget.load_example()
solve_mass_balance() Algoritmo de Kahn para cálculo topológico de vazões
node_added (sinal) Emitido com node_id ao adicionar equipamento
connection_made (sinal) Emitido com (source, target) ao criar conexão
BPMN 2.0
from proeng.modules.bpmn import BPMNAutoWidget widget = BPMNAutoWidget() widget.zoom_in() # +15% widget.update_zoom(1.5) # zoom direto widget.draw_diagram() # redesenha # shapes: "Tarefa", "Gateway Exclusivo (X)", # "Evento Início", "Subprocesso" ...
nodes dict {id: {text, shape, level, lane, children, parent}}
zoom range 0.3 a 3.0 — controle preciso de visualização
Gantt — CPM
from proeng.modules.gantt import GanttWidget widget = GanttWidget() widget.add_task( name="Análise de Requisitos", start="2026-04-01", end="2026-04-15", progress=80, predecessor=None ) widget.calculate_cpm() # Critical Path
task.critical Preenchido pelo CPM — True se no caminho crítico
task.milestone Marca tarefa como marco do projeto

Requisitos mínimos: Python 3.8+ e PyQt5. Funciona em Windows 10/11, Linux e macOS. Resolução mínima 1366×768.
